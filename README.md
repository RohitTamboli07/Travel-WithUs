## 📌 About The Project

Travel With Us is a modern and responsive travel website designed to showcase travel destinations and tour packages.  

The goal of this project is to create an attractive user interface and smooth user experience using pure frontend technologies.

---

## 🚀 Features

- 🌎 Beautiful homepage with hero section
- 📍 Popular destinations section
- 📦 Travel packages display
- 🖼 Image gallery
- 📱 Fully responsive design
- 📧 Contact form
- 🎨 Smooth animations and transitions
- 🔍 Interactive navigation bar

---

## 🛠 Tech Stack

- HTML5  
- CSS3  
- JavaScript
